## 프로젝트 개요
"일본 도로 교통 표지판 번역"을 주제로 한 React 애플리케이션입니다. 일본 여행자들이 일본에서 차량을 타고 여행을 다닐 때 생소한 도로 교통 표지판을 보고 혼동하지 않도록 해석해주는 어플을 제작해보았습니다.
